HTML tags

Head tag

<head>

The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag. 

 This is a head tag

Body tag

<body>

The tag defines the document's body. The element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

This is a body tag

Heading tag

The <h1> - <h6> tags

HTML headings are titles or subtitles that you want to display on a webpage. HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading. 

 
This is a heading

Paragraph tag

<p>

The <p> tag defines a paragraph. Browsers automatically add a single blank line before and after each <p> element. 

 
This a paragraph tag

Div tag

<div>

The <div> tag defines a division or a section in an HTML document. The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The <div> tag is easily styled by using the class or id attribute. Any sort of content can be put inside the <div> tag! 

This is a div tag

Anchor tag

<a>

The anchor tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination. 

 This is an anchor tag

Image tag

<img>

The <img> tag is used to embed an image in an HTML page.
Images are not technically inserted into a web page; images are linked to web pages. The <img> tag creates a holding space for the referenced image.
The <img> tag has two required attributes:
~ src - Specifies the path to the image
~ alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed 

This is an image tag

Linebreak tag

<br>

The <br> tag inserts a single line break. The <br> tag is useful for writing addresses or poems. The <br> tag is an empty tag which means that it has no end tag. 


this is a 
 linebreak

Bold tag

<b>

The HTML <b> element defines bold text, without any extra importance.

This is a bold tag

Italic tag

<i>

The <i> tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.

This is an italic tag
